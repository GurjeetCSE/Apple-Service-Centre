# 🍎 Apple Service Centre - Full-Stack Web Application

A premium, full-stack web platform designed for an Apple Service Centre. This application features a modern "Glassmorphism" UI, secure user authentication, interactive repair booking, and a powerful administrative dashboard with real-time analytics.

## 🚀 Live Demo
The application is deployed on Vercel: [Live Site](https://apple-service-centre.vercel.app/)

## ✨ Key Features

### 👤 User Features
- **Secure Authentication**: Signup/Login with Bcrypt password hashing.
- **Password Strength Meter**: Real-time feedback during account creation.
- **User Dashboard**: Manage profiles and view complete service history.
- **Repair Booking**: Seamless workflow to request repairs for iPhone, Mac, iPad, and Watch.
- **Real-time Notifications**: In-app alerts for status updates (e.g., "In Progress" -> "Completed").
- **Customer Reviews**: Add and view star-rated feedback.

### 🛡️ Admin Features
- **Management Dashboard**: Overview of all repair requests and customer inquiries.
- **Data Visualization**: Interactive Chart.js doughnut charts showing device distribution.
- **Status Control**: Real-time status updates for repairs.
- **Inquiry Inbox**: Dedicated section to manage contact form messages.

### 🎨 UI/UX Design
- **Premium Glassmorphism**: Stunning blur effects and sleek dark mode aesthetics.
- **Fluid Transitions**: Cinematic page fade-in and slide animations.
- **Skeleton Loaders**: Industry-standard loading states for a fast perceived performance.
- **Fully Responsive**: Optimized for mobile, tablet, and desktop.

## 🛠️ Technology Stack
- **Frontend**: HTML5, Vanilla CSS3, Javascript (ES6+)
- **Backend**: Node.js (Vercel Serverless Functions)
- **Database**: SQLite (via `sql.js`)
- **Libraries**: Chart.js (Data Vis), Bcrypt.js (Security), FontAwesome (Icons)

## 📦 Getting Started Locally

1. **Clone the repository**:
   ```bash
   git clone https://github.com/GurjeetCSE/Apple-Service-Centre.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the local server**:
   ```bash
   node server.js
   ```

4. **Open in Browser**:
   Visit `http://localhost:3000`
